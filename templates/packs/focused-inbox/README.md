# Focused Inbox

Use this pack to turn noisy sources into a small local queue that agents can
help with safely.

The point is not to mirror every message. The point is to keep:

- decisions that need a response
- people waiting on you
- high-signal commitments
- facts worth remembering

Everything is private by default.

Suggested future sources, after connector use cases are reviewed:

- Notion for explicit work queues.
- Linear for product commitments.
- Slack or email only after strong filters and review policy exist.
